### 语法1，小李是中国人，这里的は都读wa
* 1， A は B です、A是B
> 私は　学生です，我是学生
* 2， A は B ですか、A是B吗？
> 你は　学生ですか，你是学生吗？
* 3， A は B ではありません、A不是B
> 私は　学生　でわあぃません，我不是学生
* 4， A の　B、A的B
> 私　の　猫，我的猫
### 语法2，这是书，事物指示代词
* 1， これ / それ / あれ  は B です，这个/那个/那个 是 B
  *  これ，这个，双方相对则离说话人近，同向则离两人都近
  *  それ，那个，双方相对则离听话人近，同向则离双方较远
  *  あれ，那个，双方相对或同向都更远
> これ　は　猫です，这是猫
* 2、誰ですか　/  何ですか，是谁啊？/ 是什么啊？
  *  誰，发音：だれ，用于不知到什么人
     * 誰用于比自己地位低于的,どなた用于比自己地位高，年老的人
  *  何，发音：なん，用于不知道什么事物
> これ　は　何ですか，这是什么啊？
* 3， この /  その / あの  A は  B です，这个/那个/那个A 是 B，用法和1一样，但是这三个指示代词后必须接名词
> この猫 は 誰のですか，这只猫是谁的？   誰の后面省略了宾语 猫，应该是誰の猫
* 4， どれ / どの 名词，  哪个？ 用法和1，3一样，どれ单独使用，どの必须接名词
> 私の猫 は どれですか,我的猫是哪一只呢？
> 
> 私の猫 は どの猫 ですか,我的猫是哪一只呢？
### 语法3，这里是商场，场所指示代词
* 1， ここ / そこ / あそこ  は B です，这里/那里/那里 是 B，用法和事物指示代词一样
> ここは 图书馆 です，这里是图书馆
* 2, Aは B（这个名词必须是场所）です，第二个名词表示场所，表示A在B场所
> 老师 は 事务所です，猫在事务所
* 3, Aは どこですか，A在哪里呢？
> 厕所は　どこですか，厕所在哪里呢？　
* 4， Aも B です，A也是B
> 私も 学生です，我也是学生
* 5, Aは Bですか ，Cですか，A是B还是C呢？后面还能再加
> あなたは 日本人ですか，中国人ですか，你是日本人还是中国人呢？
* 6， Aは いくらですか，A多少钱？ 
> この猫 はいくらですか，这只猫多少钱？
### 语法4，存在句子
* 1，名［场所］に 名［事物］が あります / います，某地有某物
* 2，名［事物］に 名［场所］にあります / います，某地有某物
  * あります，表示存在，用于没有意志的事物，植物，家具，车之类的
  * います，表示存在，用于有意志的事物：人，动物等
* 3，A と　B，A和B，助词：と表示并列关系，相当于和
